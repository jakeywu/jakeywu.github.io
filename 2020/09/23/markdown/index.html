发表于2020-09-23更新于2020-09-23

Markdown是一种纯文本格式的标记语言, 通过约定的语法使普通文本具备一定格式。 因此，只要支持Markdown的平台都可以达成同样的效果，让作者专心码字。Hexo已默认支持Markdown renderer, 用户只需要掌握Markdown语法即可。
